# Assets Folder

Place your logo files in this directory:

```
assets/
├── undiksha.png              (Universitas Pendidikan Ganesha)
└── sistem_informasi.png      (Sistem Informasi UNDIKSHA)
```

**See parent directory's LOGO_SETUP.md for detailed setup instructions.**

---

## Quick Start

1. Place logo files here (`undiksha.png` and `sistem_informasi.png`)
2. Open `streamlit_app.py`
3. Find the LOGO CONFIGURATION section
4. Change `SHOW_LOGOS = False` to `SHOW_LOGOS = True`
5. Restart Streamlit

That's it! The logos will appear in the sidebar. ✓
